TrackPro API Documentation

Integrate real-time GPS tracking, geofencing, and device management with your applications using TrackPro’s RESTful API.

Getting Started

The TrackPro API enables developers to manage GPS devices, track locations, and configure alerts programmatically. All requests use HTTPS and return JSON with UTF-8 encoding.

Base URL

https://api.trackpro.arybit.co.ke/v1

Authentication

Use a Bearer token in the Authorization header. Get your API key from the TrackPro Dashboard.

Authorization: Bearer your_api_key

Note: Tokens expire after 2 hours. Refresh every 90 minutes to avoid interruptions.

Rate Limits

100 requests per minute per IP. Contact support@trackpro.arybit.co.ke to request an increase.

API Endpoints

Core endpoints for interacting with TrackPro. Responses include a code (0 for success, non-zero for errors) and a record or message field.

Get Access Token

Generate a new access token for authentication.

  • Endpoint: POST /authorization
  • Parameters:
    • account (string, required): Your TrackPro username.
    • time (integer, required): Current UNIX timestamp.
    • signature (string, required): MD5(MD5(password) + time).

Example Request:

const axios = require('axios');
const crypto = require('crypto');

const account = 'your_username';
const password = 'your_password';
const time = Math.floor(Date.now() / 1000);
const signature = crypto.createHash('md5')
  .update(crypto.createHash('md5').update(password).digest('hex') + time)
  .digest('hex');

axios.post('https://api.trackpro.arybit.co.ke/v1/authorization', {
  account,
  time,
  signature
}).then(response => {
  console.log('Access Token:', response.data.record.access_token);
}).catch(error => {
  console.error('Error:', error.response.data);
});

Example Response:

{
  "code": 0,
  "record": {
    "access_token": "your_api_key",
    "expires_in": 7200
  }
}

Get Latest Device Coordinates

Retrieve the latest GPS coordinates for one or more devices.

  • Endpoint: GET /tracking/latest
  • Parameters:
    • access_token (string, required): Bearer token.
    • imeis (string, required): Comma-separated device IMEIs.

Example Request:

axios.get('https://api.trackpro.arybit.co.ke/v1/tracking/latest', {
  params: { access_token: 'your_api_key', imeis: '123456789012345,678901234567890' },
  headers: { 'Authorization': 'Bearer your_api_key' }
}).then(response => {
  console.log('Tracking Data:', response.data.record);
}).catch(error => {
  console.error('Error:', error.response.data);
});

Example Response:

{
  "code": 0,
  "record": [
    {
      "imei": "123456789012345",
      "latitude": -1.286389,
      "longitude": 36.817223,
      "speed": 60,
      "course": 195,
      "gpstime": 1697059200,
      "servertime": 1697059500,
      "battery": 100
    }
  ]
}

Send Device Command

Send commands to devices (e.g., enable/disable tracking, engine cutoff).

  • Endpoint: POST /command/send
  • Parameters:
    • access_token (string, required): Bearer token.
    • imei (string, required): Device IMEI.
    • command (string, required): Command (e.g., RELAY,1).

Example Request:

axios.post('https://api.trackpro.arybit.co.ke/v1/command/send', {
  access_token: 'your_api_key',
  imei: '123456789012345',
  command: 'RELAY,1'
}, {
  headers: { 'Authorization': 'Bearer your_api_key' }
}).then(response => {
  console.log('Command ID:', response.data.record.command_id);
}).catch(error => {
  console.error('Error:', error.response.data);
});

Example Response:

{
  "code": 0,
  "record": {
    "command_id": "cmd_123456",
    "status": "Sent"
  }
}

Note: Poll /command/status every 1 second (up to 30 times) to check status.

Create Geofence

Create a circular geofence for a device.

  • Endpoint: POST /geofence/create
  • Parameters:
    • access_token (string, required): Bearer token.
    • imei (string, required): Device IMEI.
    • geofencename (string, required): Geofence name.
    • alarmtype (integer, required): 0 (enter), 1 (exit), 2 (both).
    • longitude (float, required): Center longitude.
    • latitude (float, required): Center latitude.
    • radius (integer, required): Radius in meters.

Example Request:

axios.post('https://api.trackpro.arybit.co.ke/v1/geofence/create', {
  access_token: 'your_api_key',
  imei: '123456789012345',
  geofencename: 'home',
  alarmtype: 2,
  longitude: 36.817223,
  latitude: -1.286389,
  radius: 300
}, {
  headers: { 'Authorization': 'Bearer your_api_key' }
}).then(response => {
  console.log('Geofence Created:', response.data.record);
}).catch(error => {
  console.error('Error:', error.response.data);
});

Example Response:

{
  "code": 0,
  "record": {
    "geofence_id": "gf_123456",
    "geofencename": "home",
    "status": "Created"
  }
}

Get Device List

Retrieve devices associated with your account (max 500 devices).

  • Endpoint: GET /devices
  • Parameters:
    • access_token (string, required): Bearer token.

Example Request:

axios.get('https://api.trackpro.arybit.co.ke/v1/devices', {
  params: { access_token: 'your_api_key' },
  headers: { 'Authorization': 'Bearer your_api_key' }
}).then(response => {
  console.log('Devices:', response.data.record);
}).catch(error => {
  console.error('Error:', error.response.data);
});

Example Response:

{
  "code": 0,
  "record": [
    {
      "imei": "123456789012345",
      "devicename": "Vehicle_001",
      "devicetype": "Teltonika FMB920",
      "onlinetime": 1697059200

Error Handling

Handle errors based on the code field in responses.

Code HTTP Status Description
0200Success
1401Invalid or expired token
2400Invalid parameters
3429Rate limit exceeded
4404Resource not found

Example Error Response:

{
  "code": 1,
  "message": "Invalid or expired access token"
}

Best Practices

  • Token Management: Refresh tokens every 90 minutes to avoid interruptions.
  • Rate Limiting: Use exponential backoff for 429 errors.
  • Data Validation: Validate IMEIs and coordinates before requests.
  • Logging: Log all API responses for debugging.
  • Secure Storage: Store API keys securely, avoiding hardcoding.
  • Webhook Support: Configure webhooks for real-time updates (contact support).
